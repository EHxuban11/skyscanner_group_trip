# Pomodoro Group Trip

## Setup
```bash
# install root & client deps
npm install

# install server deps
cd server
npm install

# back to root
cd ..

# run first migration (SQLite dev.db)
npx prisma migrate dev --name init

# generate Prisma client
npx prisma generate

npx prisma studio
